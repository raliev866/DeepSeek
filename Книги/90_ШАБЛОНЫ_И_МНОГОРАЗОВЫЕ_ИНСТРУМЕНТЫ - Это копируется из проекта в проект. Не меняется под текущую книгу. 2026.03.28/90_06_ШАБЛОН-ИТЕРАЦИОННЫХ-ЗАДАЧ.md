# 90_06_ШАБЛОН-ИТЕРАЦИОННЫХ-ЗАДАЧ.md

## НАЗНАЧЕНИЕ
Этот документ определяет **поэтапный план работы над книгой** в режиме «Архитектор информации». Каждый этап фокусируется на одном типе задач, что позволяет системно подойти к созданию качественного продукта.

**Источник стандартов:** Все требования к формату, структуре глав, стилю изложения зафиксированы в документе:
- `00. Задачи. ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ТЗ) НА НАПИСАНИЕ КНИГИ.md`

**Связанные шаблоны:**
- `90_00_Шаблон_№0_«ПАСПОРТ-КНИГИ»` — цели и структура книги
- `90_01_Шаблон_№1_СТРУКТУРА_ГЛАВЫ` — стандарт каждой главы
- `90_02_Шаблон_№2_СТРУКТУРА_ТОМА` — стандарт тома
- `90_03_Шаблон_№3_СТРУКТУРА_РАЗДЕЛА` — стандарт раздела
- `90_05_Шаблон_№5_Формат_текста` — правила оформления

---

## 📌 ЭТАП 1. АРХИТЕКТУРА И СТРУКТУРА
*Цель: создать логичную, непротиворечивую структуру книги, определить все элементы.*

### Задачи:
- [ ] Создать общую структуру книги (тома, разделы, главы)
- [ ] Определить названия всех томов, разделов, глав
- [ ] Создать Карту проекта (00_01_КАРТА_ПРОЕКТА.md)
- [ ] Создать Синопсис и оглавление (00_02_СИНОПСИС-И-ОГЛАВЛЕНИЕ.md)
- [ ] Создать Паспорт экспертизы (00_03_ПАСПОРТ-ЭКСПЕРТИЗЫ.md)
- [ ] **Структурная оптимизация:**
  - [ ] Выявить и удалить все маленькие главы, их содержимое присоединить к другим главам
  - [ ] Проверить логичность последовательности глав, разделов, томов
  - [ ] Исправить все нелогичности в структуре
- [ ] **Нумерация и именование:**
  - [ ] Установить единую нумерацию глав, разделов, томов
  - [ ] Переименовать файлы и папки в соответствии с новой нумерацией
  - [ ] Создать навигационные ссылки между главами

---

## 📌 ЭТАП 2. НАПИСАНИЕ ЧЕРНОВИКОВ
*Цель: быстро создать полный черновик книги, не отвлекаясь на детали.*

### Задачи:
- [ ] Написать черновики всех глав (по одному файлу на главу)
- [ ] Следовать базовой структуре из ТЗ (без углублённой проработки)
- [ ] Создать черновики приложений (глоссарий, словарь сокращений, сборник химических соединений)
- [ ] Фиксировать в TODO незавершённые элементы и вопросы

---

## 📌 ЭТАП 3. РЕДАКТУРА И УНИФИКАЦИЯ
*Цель: привести все главы к единому стандарту, устранить повторы и нелогичности.*

### Задачи:
- [ ] **Приведение к стандартам (из ТЗ и шаблонов):**
  - [ ] Проверить каждую главу на соответствие шаблону №1 (структура главы)
  - [ ] Проверить каждый том на соответствие шаблону №2 (структура тома)
  - [ ] Проверить каждый раздел на соответствие шаблону №3 (структура раздела)
  - [ ] Привести оформление в соответствие с шаблоном №5 (формат текста)
- [ ] **Устранение дублирования:**
  - [ ] Найти и удалить повторяющуюся информацию (межглавные повторы)
  - [ ] Объединить повторяющиеся объяснения с перекрёстными ссылками
- [ ] **Логическая вычитка:**
  - [ ] Исправить все нелогичности в изложении
  - [ ] Проверить причинно-следственные связи
  - [ ] Убедиться, что каждая глава имеет чёткую связь с предыдущей и следующей
- [ ] **Добавление недостающих элементов:**
  - [ ] Добавить вопросы о расходовании веществ в каждую главу (тип В в блоке вопросов)
  - [ ] Убедиться, что в метаболическом уровне каждой главы описан блок «Расходование веществ»
  - [ ] Добавить практические задания для читателя
  - [ ] Добавить аудиоверсии для всех таблиц (если планируется аудиокнига)

## 📌 ЭТАП 4. ДЕТАЛИЗАЦИЯ И ДОРАБОТКА
*Цель: углубить содержание, добавить иллюстрации, кейсы, приложения.*

### Задачи:
- [ ] **Углубление содержания:**
  - [ ] Добавить клинические примеры и кейсы в каждую главу
  - [ ] Расширить практические рекомендации
  - [ ] Добавить контрольные вопросы (3–10 в зависимости от сложности главы)
- [ ] **Визуализация:**
  - [ ] Создать или добавить схемы, диаграммы, иллюстрации
  - [ ] Убедиться, что для каждой сложной схемы есть текстовое описание (для аудио)
- [ ] **Приложения:**
  - [ ] Завершить глоссарий терминов
  - [ ] Завершить словарь сокращений
  - [ ] Завершить сборник химических соединений с объяснениями
  - [ ] Завершить список литературы и источников
  - [ ] Завершить ответы на частые вопросы

## 📌 ЭТАП 5. ФИНАЛЬНАЯ ВЫЧИТКА И ПОДГОТОВКА
*Цель: подготовить книгу к публикации (или к следующему этапу — экспертизе).*

### Задачи:
- [ ] **Финальная вычитка:**
  - [ ] Орфография, пунктуация, стилистика
  - [ ] Проверка всех ссылок (внутренних и внешних)
  - [ ] Проверка всех терминов на единообразие
- [ ] **Форматирование:**
  - [ ] Переписать оглавление под финальную нумерацию и расположение глав
  - [ ] Перезаписать названия файлов и папок, если были изменения
  - [ ] Убедиться, что все файлы имеют правильное обрамление (``` в начале и конце)
  - [ ] Проверить, что все заголовки имеют правильный уровень вложенности
- [ ] **Сборка:**
  - [ ] Собрать все тома в единую структуру
  - [ ] Создать итоговое оглавление с гиперссылками
  - [ ] Создать маршрутную карту читателя (если не создана)
- [ ] **Подготовка к экспертизе:**
  - [ ] Сформировать запрос на экспертизу (по Паспорту экспертизы)
  - [ ] Подготовить список вопросов для эксперта
