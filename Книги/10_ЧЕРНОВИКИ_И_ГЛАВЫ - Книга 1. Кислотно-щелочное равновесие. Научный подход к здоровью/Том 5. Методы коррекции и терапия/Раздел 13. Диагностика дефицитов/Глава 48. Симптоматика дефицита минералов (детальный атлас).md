# Глава 48. Симптоматика дефицита минералов (детальный атлас)

**Том 5. Методы коррекции и терапия**  
**Раздел 13. Диагностика дефицитов**

---

### Аннотация

Дефициты минералов — не просто лабораторные отклонения. Это конкретные симптомы, которые человек чувствует каждый день: судороги, усталость, бессонница, выпадение волос. Хронический ацидоз усугубляет эти дефициты, а дефициты, в свою очередь, усиливают ацидоз. В этой главе мы создадим «атлас» симптомов — чтобы вы могли распознать признаки нехватки ключевых минералов и понять, с чего начинать коррекцию.

---

### Ключевые понятия главы

- **Микроэлементы** — минералы, необходимые в малых количествах (цинк, селен, медь, марганец).
- **Макроэлементы** — минералы, необходимые в больших количествах (кальций, магний, калий, натрий, фосфор).
- **Кофактор** — вещество, необходимое для работы фермента.
- **Симптомокомплекс** — устойчивая совокупность симптомов, характерная для определённого дефицита.

---

## 1. Почему дефициты минералов критичны при ацидозе

Минералы участвуют во всех аспектах кислотно-щелочного равновесия:

| Роль | Примеры минералов |
|------|-------------------|
| **Буферные системы** | Кальций, фосфор, магний, бикарбонат (в форме солей) |
| **Работа ферментов** | Магний, цинк, марганец, молибден (кофакторы) |
| **Кислородный обмен** | Железо, медь |
| **Нервная проводимость** | Калий, натрий, кальций, магний |
| **Костная ткань** | Кальций, фосфор, магний |
| **Мышечное сокращение** | Кальций, магний, калий |

При хроническом ацидозе:
- **Повышается выведение** минералов с мочой (кальций, магний, калий).
- **Нарушается всасывание** (из-за дисбиоза, нарушения pH в кишечнике).
- **Увеличивается потребность** (из-за работы буферных систем).

**Результат:** дефициты развиваются быстрее, а их симптомы усугубляют течение ацидоза.

## 2. Магний (Mg²⁺)

### 2.1. Роль в организме

Магний — кофактор более 300 ферментов. Участвует в:
- Энергетическом обмене (синтез АТФ).
- Работе Na⁺/K⁺-АТФазы (поддержание клеточного потенциала).
- Синтезе белка и ДНК.
- Регуляции мышечного тонуса (антагонист кальция).
- Синтезе нейромедиаторов (серотонин, ГАМК).
- Активации витамина D.

### 2.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Нервно-мышечная** | Мышечные судороги (икры, стопы), подёргивания век, тремор, «синдром беспокойных ног», онемение, покалывание |
| **Сердечно-сосудистая** | Сердцебиение, экстрасистолия, повышенное давление, спазм коронарных артерий |
| **Нервная система** | Тревога, раздражительность, бессонница, депрессия, головные боли, повышенная чувствительность к шуму |
| **Обмен веществ** | Усталость, слабость, инсулинорезистентность, склонность к запорам (спазм гладкой мускулатуры) |
| **Костная система** | Остеопороз, дефицит витамина D (магний нужен для его активации) |
| **Менструальный цикл** | Предменструальный синдром (ПМС), спастические боли, нарушение цикла |

**Характерные признаки:**
- **Судороги в икрах по ночам** — классический признак.
- **«Синдром беспокойных ног»** — желание двигать ногами в покое, особенно вечером.
- **Запоры** — спазм гладкой мускулатуры кишечника.

### 2.3. Факторы риска

- Кофе, чай (усиливают выведение).
- Алкоголь.
- Диуретики, ингибиторы протонной помпы (омепразол).
- Хронический стресс (повышает потерю магния).
- Высокое потребление кальция без магния.
- Заболевания кишечника (синдром мальабсорбции).

## 3. Калий (K⁺)

### 3.1. Роль в организме

Калий — главный внутриклеточный катион. Участвует в:
- Поддержании мембранного потенциала.
- Регуляции сердечного ритма.
- Мышечном сокращении (включая сердечную мышцу).
- Передаче нервных импульсов.
- Поддержании кислотно-щелочного равновесия (обмен на H⁺ в почках).

### 3.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Нервно-мышечная** | Мышечная слабость, вялость, утомляемость, параличи (при тяжёлом дефиците) |
| **Сердечно-сосудистая** | Сердцебиение, аритмии (экстрасистолия), снижение давления, изменения на ЭКГ |
| **Желудочно-кишечная** | Запоры, вздутие, паралитическая кишечная непроходимость (при тяжёлом дефиците) |
| **Общие** | Усталость, апатия, полиурия (частое мочеиспускание) |

**Характерные признаки:**
- **Слабость в ногах** после физической нагрузки или к вечеру.
- **Сердцебиение** в покое или при незначительной нагрузке.
- **Жажда и частое мочеиспускание** (особенно при сочетании с дефицитом магния).

### 3.3. Факторы риска

- Диуретики (особенно петлевые и тиазидные).
- Слабительные (злоупотребление).
- Рвота, диарея.
- Высокое потребление натрия (соль).
- Алкоголь.
- Гипомагниемия (магний необходим для удержания калия в клетках).

## 4. Кальций (Ca²⁺)

### 4.1. Роль в организме

Кальций — главный минерал костной ткани. Участвует в:
- Формировании и поддержании костей и зубов.
- Мышечном сокращении.
- Нервной проводимости.
- Свёртывании крови.
- Работе ферментов.

### 4.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Костная** | Остеопороз, остеопения, частые переломы, боли в костях |
| **Нервно-мышечная** | Судороги, парестезии (онемение, покалывание), тетания (спазм кисти — «рука акушера»), спазм лицевых мышц |
| **Зубы** | Кариес, пародонтит, хрупкость зубов |
| **Сердечно-сосудистая** | Аритмии, нарушение сократимости миокарда |
| **Кожа, волосы, ногти** | Сухость кожи, ломкость ногтей, выпадение волос |

**Характерные признаки:**
- **Судороги** (отличаются от магниевых: кальциевые судороги чаще в покое, магниевые — после нагрузки).
- **Симптом Хвостека** — подёргивание мышц лица при постукивании по лицевому нерву (у нижней челюсти).
- **Симптом Труссо** — спазм кисти при пережатии плеча манжетой тонометра.

### 4.3. Факторы риска

- Дефицит витамина D.
- Гипопаратиреоз.
- Хроническая болезнь почек (нарушение активации витамина D).
- Длительный приём ингибиторов протонной помпы (снижают всасывание).
- Высокое потребление фосфатов (газировка, переработанные продукты).

## 5. Цинк (Zn²⁺)

### 5.1. Роль в организме

Цинк — кофактор более 300 ферментов. Участвует в:
- Иммунной функции (Т-лимфоциты, NK-клетки).
- Заживлении ран.
- Синтезе белка и ДНК.
- Вкусе и обонянии.
- Функции щитовидной железы.
- Антиоксидантной защите.

### 5.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Иммунитет** | Частые инфекции (простуды, ОРВИ), затяжное течение, хроническое воспаление |
| **Кожа** | Плохое заживление ран, акне, дерматиты, сухость, шелушение |
| **Волосы, ногти** | Выпадение волос, ломкость ногтей, белые пятна на ногтях (лейконихия) |
| **Вкус, обоняние** | Извращение вкуса, снижение чувствительности, потеря аппетита |
| **Репродуктивная** | Снижение либидо, бесплодие (нарушение сперматогенеза), нарушения менструального цикла |
| **Нервная система** | Нарушение когнитивных функций, депрессия, задержка роста у детей |
| **Зрение** | Нарушение сумеречного зрения (цинк участвует в метаболизме витамина A) |

**Характерные признаки:**
- **Белые пятна на ногтях** — один из самых частых визуальных признаков.
- **Частые простуды** — особенно если длятся более 7–10 дней.
- **Медленное заживление** царапин и порезов.

### 5.3. Факторы риска

- Вегетарианство/веганство (фитаты в злаках снижают всасывание цинка).
- Заболевания кишечника (болезнь Крона, целиакия).
- Алкоголизм.
- Длительный стресс (повышенное выведение).
- Беременность и лактация.

---

## 6. Железо (Fe²⁺/Fe³⁺)

### 6.1. Роль в организме

Железо — ключевой компонент гемоглобина и миоглобина. Участвует в:
- Транспорте кислорода.
- Клеточном дыхании (цитохромы).
- Синтезе ДНК.
- Функции иммунитета.

### 6.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Общие** | Усталость, слабость, снижение работоспособности, бледность кожи и слизистых |
| **Нервная** | Головокружение, головные боли, снижение концентрации, «туман» в голове |
| **Сердечно-сосудистая** | Сердцебиение, одышка при нагрузке, тахикардия |
| **Кожа, волосы, ногти** | Сухость кожи, ломкость волос, выпадение волос, ложкообразные ногти (койлонихия) |
| **Специфические** | Извращение вкуса (pica) — желание есть мел, землю, лёд; ангулярный стоматит («заеды»); синдром беспокойных ног |

**Характерные признаки:**
- **«Заеды»** — трещины в уголках рта.
- **Синдром беспокойных ног** — часто связан с дефицитом железа (и магния).
- **Желание жевать лёд** (пагофагия) — специфический признак железодефицита.

### 6.3. Факторы риска

- Менструирующие женщины (особенно с обильными месячными).
- Вегетарианство/веганство (гемовое железо из мяса усваивается лучше).
- Заболевания ЖКТ (гастрит, целиакия, болезнь Крона).
- Донорство.
- Длительный приём ингибиторов протонной помпы (снижают всасывание).

## 7. Селен (Se)

### 7.1. Роль в организме

Селен — компонент антиоксидантных ферментов (глутатионпероксидаза). Участвует в:
- Защите от окислительного стресса.
- Функции щитовидной железы (дейодиназы).
- Иммунной функции.
- Репродуктивной функции.

### 7.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Щитовидная железа** | Гипотиреоз (особенно при аутоиммунном тиреоидите), увеличение щитовидной железы |
| **Иммунитет** | Снижение иммунитета, частые инфекции |
| **Мышцы** | Мышечная слабость, боли, кардиомиопатия (болезнь Кешана — редкая) |
| **Кожа, волосы, ногти** | Сухость кожи, ломкость волос, ломкость ногтей |
| **Репродуктивная** | Снижение фертильности у мужчин и женщин |

**Характерные признаки:**
- **Слабость и мышечные боли** без видимой причины.
- **Склонность к гипотиреозу** или трудности в подборе дозы тироксина.
- **Частые инфекции** на фоне нормальных анализов крови.

### 7.3. Факторы риска

- Проживание в регионах с бедными селеном почвами (многие регионы России, Европы).
- Заболевания кишечника (мальабсорбция).
- Диализ (потеря селена).
- Веганство.

---

## 8. Йод (I)

### 8.1. Роль в организме

Йод — компонент тиреоидных гормонов (Т3, Т4). Участвует в:
- Регуляции основного обмена.
- Росте и развитии.
- Термогенезе.

### 8.2. Симптомы дефицита

| Система | Симптомы |
|---------|----------|
| **Щитовидная железа** | Зоб (увеличение железы), гипотиреоз, усталость, зябкость, запоры, набор веса |
| **Когнитивные** | Снижение памяти, внимания, замедление мышления |
| **Репродуктивные** | Нарушение менструального цикла, бесплодие |
| **У детей** | Задержка физического и умственного развития (кретинизм — при тяжёлом дефиците) |

**Характерные признаки:**
- **Зябкость** — мёрзнут даже в тёплом помещении.
- **Запоры** на фоне общей вялости.
- **Увеличение щитовидной железы** (видимое или пальпируемое).

### 8.3. Факторы риска

- Проживание в регионах с бедными йодом почвами (эндемический зоб).
- Отказ от йодированной соли.
- Потребление гойтрогенов (капуста, соя, брокколи) в больших количествах (блокируют захват йода).
- Беременность и лактация (повышенная потребность).

## 9. Дифференциальная диагностика: как не перепутать

Многие симптомы дефицитов пересекаются. Вот ключевые различия:

| Симптом | Возможные дефициты | Как различить |
|---------|-------------------|---------------|
| **Судороги в икрах** | Магний, кальций, калий | Магний — чаще после нагрузки, кальций — в покое ночью, калий — сопровождается слабостью |
| **Сердцебиение** | Калий, магний, кальций | При дефиците калия — часто с полиурией; при дефиците магния — с тревогой, судорогами |
| **Усталость** | Железо, магний, калий, селен | Железо — с бледностью; магний — с судорогами; калий — со слабостью в ногах; селен — с мышечными болями |
| **Выпадение волос** | Железо, цинк, селен | Железо — с бледностью; цинк — с белыми пятнами на ногтях; селен — с мышечной слабостью |
| **Синдром беспокойных ног** | Железо, магний | Железо — часто с извращением вкуса; магний — с тревогой, судорогами |

## 10. Что делать при подозрении на дефицит

### 10.1. Первый шаг: лабораторная диагностика

Симптомы — это сигнал, но не диагноз. Для подтверждения нужны анализы (см. Главу 49).

### 10.2. Второй шаг: коррекция питания

Прежде чем принимать добавки, оптимизируйте рацион:
- **Магний:** зелень, орехи, семена, бобовые, какао.
- **Калий:** овощи (картофель, томаты, тыква), зелень, авокадо, бананы.
- **Кальций:** кунжут, зелень, миндаль, сардины, молочные продукты (при переносимости).
- **Цинк:** устрицы, тыквенные семечки, орехи, мясо, печень.
- **Железо:** красное мясо, печень, шпинат, бобовые (с витамином C для усвоения).
- **Селен:** бразильские орехи, тунец, сардины, печень.
- **Йод:** морская капуста, рыба, йодированная соль.

### 10.3. Третий шаг: добавки при необходимости

Если питание не покрывает потребность (или есть нарушения всасывания), могут потребоваться добавки. Выбор формы (цитрат, глицинат, пиколинат и др.) важен для усвоения и переносимости (см. Главу 55).

### 10.4. Четвёртый шаг: мониторинг

После начала коррекции важно отслеживать:
- Уменьшение симптомов (ведём дневник).
- Лабораторные показатели (через 2–3 месяца).
- Возможную передозировку (особенно для железа, селена, йода).

### Резюме для читателя

**Дефициты минералов — не абстракция. Они проявляются конкретными симптомами, которые можно распознать.**

| Минерал | Ключевые симптомы |
|---------|-------------------|
| **Магний** | Судороги, тревога, бессонница, запоры |
| **Калий** | Слабость, сердцебиение, полиурия |
| **Кальций** | Судороги в покое, остеопороз, онемение |
| **Цинк** | Белые пятна на ногтях, частые простуды, плохое заживление |
| **Железо** | Усталость, бледность, «заеды», желание жевать лёд |
| **Селен** | Мышечная слабость, склонность к гипотиреозу |
| **Йод** | Зябкость, запоры, увеличение щитовидной железы |

**Что важно:**
1. **Симптомы пересекаются** — ориентируйтесь на комплекс признаков.
2. **Диагноз требует подтверждения** — анализы крови (см. Главу 49).
3. **Коррекция начинается с питания** — добавки не заменяют здоровый рацион.
4. **Форма имеет значение** — выбирайте биодоступные формы минералов.
5. **Дефициты часто сочетаются** — особенно магний и калий, цинк и селен.

Помните: ваше тело посылает сигналы. Научитесь их слышать — и вы сможете вовремя скорректировать то, что мешает вашему здоровью и кислотно-щелочному равновесию.
