## 2. ПРАВИЛА ОФОРМЛЕНИЯ

### 2.1. Заголовки

| Уровень | Формат | Пример |
|---------|--------|--------|
| **Заголовок 1** | `# Текст` | `# Глава 1. Что такое pH` |
| **Заголовок 2** | `## X.1. Текст` | `## 1.1. Определение pH` |
| **Заголовок 3** | `### X.1.1. Текст` | `### 1.1.1. Шкала pH` |
| **Заголовок 4** | `#### X.1.1.1. Текст` | `#### 1.1.1.1. Крайние значения` |

**Важно:** Заголовки 2-го уровня и ниже всегда начинаются с номера главы и раздела (например, `## 5.3.` для третьего раздела пятой главы).

### 2.2. Текст

- **Абзацы** отделяются пустой строкой
- **Длина абзаца:** 3-7 предложений (оптимально для чтения)
- **Переносы слов:** не использовать принудительные переносы

### 2.3. Выделения

| Назначение | Формат | Пример |
|------------|--------|--------|
| **Важное** | `**текст**` | `**pH крови** строго регулируется` |
| *Термин при первом упоминании* | `*текст*` | `*Буферная ёмкость* — это...` |
| ***Особо важное*** | `***текст***` | `***Никогда*** не превышайте дозу` |

### 2.4. Списки

**Маркированный список:**
```
- Пункт первый
- Пункт второй
  - Подпункт (с отступом 2 пробела)
  - Ещё подпункт
- Пункт третий
```

**Нумерованный список:**
```
1. Первый пункт
2. Второй пункт
   1. Подпункт 2.1 (с отступом 3 пробела)
   2. Подпункт 2.2
3. Третий пункт
```

### 2.5. Таблицы

```markdown
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|-------------|-------------|-------------|
| Ячейка 1    | Ячейка 2    | Ячейка 3    |
| Ячейка 4    | Ячейка 5    | Ячейка 6    |
```

**Правила оформления таблиц:**
- Всегда использовать выравнивание по левому краю (по умолчанию)
- Для чисел с плавающей точкой использовать формат `X.X`
- В первой строке — заголовки с большой буквы
- Разделительная строка обязательна (`|---|---|---|`)

### 2.6. Цитаты и выделенные блоки

```
> Это цитата или важное примечание.
> Она может занимать несколько строк.
> Каждая строка начинается с символа >.
```

**Использовать для:**
- Важных предупреждений
- Исторических справок
- Клинических случаев

### 2.7. Код и формулы

Для формул (в тексте): `$E = mc^2$`

Для отдельных формул (отдельным блоком):
```
$$ 
\text{pH} = -\log_{10}[\text{H}^+]
$$
```

## 3. ТЕРМИНОЛОГИЯ

### 3.1. Единицы измерения

| Величина | Правильно | Неправильно |
|----------|-----------|--------------|
| миллимоль на литр | ммоль/л | ммоль\л, ммоль на литр |
| миллиэквивалент | мэкв | мЭкв, мэкв/л (если на литр) |
| водородный показатель | pH | Ph, PH |
| грамм | г | гр. |
| миллилитр | мл | мл. |

### 3.2. Написание терминов

| Термин | Правильно | Примечание |
|--------|-----------|------------|
| Кислотно-щелочной баланс | кислотно-щелочной баланс | через дефис, с маленькой в тексте |
| Кислотно-щелочное равновесие | кислотно-щелочное равновесие | через дефис |
| Потенциальная кислотная нагрузка на почки | PRAL | аббревиатура |
| Бикарбонат | бикарбонат | с маленькой |
| Лактат | лактат | с маленькой |

### 3.3. Первое упоминание термина

При первом упоминании сложного термина выделять *курсивом* и давать краткое пояснение:

```
*Ацидоз* — состояние, при котором pH крови снижается ниже 7.35.
```

## 6. ПРИМЕР ОФОРМЛЕНИЯ ГЛАВЫ

```markdown
# Глава 15. Анализ слюны

*Том 2. Диагностика и показатели | Раздел IV. Лабораторная диагностика*

---

**Краткое содержание главы:**
- Физиологические основы образования слюны
- Методика измерения pH слюны
- Интерпретация результатов
- Связь pH слюны с состоянием организма

---

## 15.1. Физиология слюнообразования

Слюна — это не просто вода. Это сложная биологическая жидкость, состав которой отражает состояние многих систем организма...

### 15.1.1. Состав слюны

Основные компоненты слюны:
- Вода (99%)
- Электролиты (натрий, калий, кальций, хлор, фосфаты, бикарбонаты)
- Ферменты (амилаза, лизоцим)
- Муцин

> **Важно!** Бикарбонаты в слюне выполняют буферную функцию, нейтрализуя кислоты, вырабатываемые бактериями зубного налёта.

---

## 15.2. Методика измерения

Для измерения pH слюны необходимо:

1. Не есть и не пить за 30 минут до измерения
2. Прополоскать рот водой
3. Набрать слюну в рот и проглотить (для стимуляции)
4. Собрать свежую порцию слюны в чистую ёмкость
5. Опустить тест-полоску на 1-2 секунды
6. Через 30 секунд сравнить с цветовой шкалой

**Нормы pH слюны:**

| Время измерения | Норма | Отклонение |
|-----------------|-------|------------|
| Утро (натощак) | 6.8-7.2 | < 6.5 или > 7.5 |
| Через 2 часа после еды | 7.0-7.5 | < 6.8 |

---

## Резюме главы

**Ключевые выводы:**
1. pH слюны отражает состояние пищеварительной системы и гидратации
2. Измерение проводится натощак или через 2 часа после еды
3. Нормальный диапазон — 6.8-7.5
4. Снижение pH слюны (< 6.5) указывает на риск кариеса и нарушение пищеварения

---

**Ключевые термины главы:**
- Слюна
- Буферная ёмкость слюны
- Лакмусовая бумага
- Гипосаливация

---

*Связь с другими главами: подробнее о диагностике в **Главе 14**, о влиянии питания в **Главе 64**.*
```

## 7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 7.1. Имена файлов

| Тип файла | Формат | Пример |
|-----------|--------|--------|
| Главы | `Глава X. Название.md` | `Глава 15. Анализ слюны.md` |
| Введения в том | `00. Введение.md` | `00. Введение.md` |
| Приложения | `Приложение N. Название.md` | `Приложение 1. Таблица PRAL.md` |
| Управляющие файлы | `NN_НАЗВАНИЕ.md` | `00_01_КАРТА_ПРОЕКТА.md` |

**Правила именования:**
- Без специальных символов (/\:?*<>|)
- Пробелы между словами (кроме управляющих файлов)
- Латиница или кириллица (в данном проекте — кириллица для глав)

### 7.2. Кодировка

- **Кодировка:** UTF-8
- **Перенос строк:** Unix (LF)
- **Конец файла:** обязательная пустая строка в конце

### 7.3. Длина строки

Рекомендуемая длина строки — **80-100 символов** для удобства чтения в редакторе, но строгого требования нет.
